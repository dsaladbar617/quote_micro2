# Quotes
## Overview

This API will provide a random motivational quote each time that it is requested.

## Request

method: GET

api: http://localhost:8080/api/v1/quote

## Response Example
```json
{
  quoteText: "",
  quoteAuthor: "",
  quoteLink: ""
}
```
